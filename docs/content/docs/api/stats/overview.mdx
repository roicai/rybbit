## Get Overview

<div className="grid grid-cols-1 xl:grid-cols-2 gap-8">
<div>

```
GET /api/overview/:site
```

Returns high-level analytics metrics for a site.

### Path Parameters

<TypeTable
  type={{
    site: {
      description: 'Site ID or site identifier',
      type: 'string | number',
      required: true,
    },
  }}
/>

### Query Parameters

Accepts all [Common Parameters](/docs/api/stats/reference#common-parameters) (time and filters).

### Response

<TypeTable
  type={{

      description: 'Total number of sessions',

    pageviews: {
      description: 'Total number of pageviews',
      type: 'number',

      description: 'Total unique users',
      type: 'number',

      description: 'Average pages viewed per session',
      type: 'number',

      description: 'Percentage of single-page sessions',

      typeDescription: 'Range: 0-100',
    },
    session_duration: {
      description: 'Average session duration in seconds',

  }}

## Get Overview (Time Series)

<div className="grid grid-cols-1 xl:grid-cols-2 gap-8">
<div>

```
GET /api/overview/:site/bucketed
```

Returns time-series analytics data broken down by configurable time buckets. Useful for charting trends over time.

### Path Parameters

<TypeTable
  type={{
    site: {
      description: 'Site ID or site identifier',
      type: 'string | number',
      required: true,
    },
  }}
/>

### Query Parameters

Accepts all [Common Parameters](/docs/api/stats/reference#common-parameters) plus the following:

<TypeTable
  type={{
    bucket: {
      description: 'Time bucket size for grouping data',
      type: 'TimeBucket',
      typeDescription: '"minute" | "five_minutes" | "ten_minutes" | "fifteen_minutes" | "hour" | "day" | "week" | "month" | "year"',
      required: true,
    },
  }}
/>

#### Bucket Selection Guide

| Bucket | Best For |
|--------|----------|
| `minute` | Real-time monitoring (last hour) |
| `five_minutes` | Short-term analysis (last few hours) |
| `hour` | Daily analysis |
| `day` | Weekly/monthly analysis |
| `week` | Monthly/quarterly analysis |
| `month` | Yearly analysis |

      type: 'number',

    users: {

      type: 'number',
    },

    },
    bounce_rate: {
      description: 'Percentage of single-page sessions',
      type: 'number',
      typeDescription: 'Range: 0-100',
    },
    session_duration: {
      description: 'Average session duration in seconds',

/>

## Get Metric

<div className="grid grid-cols-1 xl:grid-cols-2 gap-8">
<div>

```
GET /api/metric/:site
```

Returns dimensional analytics data broken down by a specific parameter. Supports pagination.

### Path Parameters

<TypeTable
  type={{
    site: {
      description: 'Site ID or site identifier',
      type: 'string | number',
      required: true,
    },
  }}
/>

### Query Parameters

Accepts all [Common Parameters](/docs/api/stats/reference#common-parameters) plus the following:

<TypeTable
  type={{
    parameter: {
      description: 'The dimension to break down by',
      type: 'FilterParameter',
      typeDescription: 'e.g., "pathname", "country", "browser", "event_name"',
      required: true,
    },
    limit: {
      description: 'Number of results per page',
      type: 'number',
      default: 100,
    },
    page: {
      description: 'Page number (1-indexed)',
      type: 'number',
      default: 1,
    },
  }}
/>

### Response

<TypeTable
  type={{
    data: {
      description: 'Array of metric results',
      type: 'MetricItem[]',
      typeDescription: 'See MetricItem structure below',

      description: 'Total number of unique values across all pages',
      type: 'number',
    },
  }}
/>

<TypeTable
  type={{

      description: 'Number of sessions for this value',

    },
    percentage: {
      description: 'Percentage of total sessions',
      type: 'number',
      typeDescription: 'Range: 0-100',
    },
    pageviews: {
      description: 'Total pageviews (for page-related parameters)',
      type: 'number',
      required: false,
    },
    bounce_rate: {
      description: 'Bounce rate percentage',
      type: 'number',
      typeDescription: 'Range: 0-100',
      required: false,

/>

## Get Live Visitors

<div className="grid grid-cols-1 xl:grid-cols-2 gap-8">
<div>

```
GET /api/live-usercount/:site
```

Returns the count of unique active sessions within the specified time window. Useful for displaying real-time visitor counts.

### Path Parameters

<TypeTable
  type={{
    site: {
      description: 'Site ID or site identifier',
      type: 'string | number',
      required: true,
    },
  }}
/>

### Query Parameters

<TypeTable
  type={{
    minutes: {
      description: 'Time window in minutes to count active sessions',
      type: 'number',
      default: 5,
      typeDescription: 'Sessions with activity within the last N minutes are considered "live"',
    },
  }}
/>

### Response

<TypeTable
  type={{
    count: {
      description: 'Number of unique active sessions',
      type: 'number',
    },
  }}
/>

#### Time Window Guide

| Minutes | Use Case |
|---------|----------|
| `1` | Very active visitors (currently browsing) |
| `5` | Standard "live" count (default) |
| `15` | Recently active visitors |
| `30` | Short-term engagement window |

</div>
<div className="flex flex-col gap-4">

<Tabs items={['cURL', 'JavaScript', 'Python']}>
  <Tab value="cURL">
```bash title="Request"
curl -X GET "https://api.rybbit.io/api/live-usercount/123?minutes=5" \
  -H "Authorization: Bearer your_api_key_here"
```
  </Tab>
  <Tab value="JavaScript">
```javascript title="Request"
const response = await fetch(
  'https://api.rybbit.io/api/live-usercount/123?minutes=5',
  {

  }

  <Tab value="Python">

    }

print(f"Live visitors: {data['count']}")
```
  </Tab>
</Tabs>

```json title="Response"

}
```

### Polling Example

```javascript title="Real-time Widget"
async function updateLiveCount() {
  const response = await fetch(
    'https://api.rybbit.io/api/live-usercount/123?minutes=5',
    {
      headers: {
        'Authorization': 'Bearer your_api_key'
      }
    }
  );
  const { count } = await response.json();
  document.getElementById('live-visitors')
    .textContent = count;
}

// Update every 15 seconds
setInterval(updateLiveCount, 15000);
```
